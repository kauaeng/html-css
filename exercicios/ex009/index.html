Listas

Listas Ordenadas

A HTML chama de ordered list todas aquelas listas onde a ordem dos itens é algo muito importante. Um passo-a-passo, por exemplo, uma lista para criar um bolo.

Para criar uma Ordered list, vamos usar a tag OL para delimitar a lista é LI(List Item) para identificar cada item da lista.
IMPORTANTE Segundo a W3C, a tag OL é de fechamento obrigatório e a tag LI o fechamento é opcional

Listas não ordenadas

Unordered list, também chamada de listas com marcadore, que são aquelas onde a ordem dos itens não influenciará no significado da lista. Ela é apenas uma ótima maneira para organizar os itens que não apresentam uma classificação necessariamente.
Para criar uma Unordered lis vamos usar a tag Ul para delimitar a lista é a tag LI para criar cada um dos seus itens internos

Existe a opção de configurar a propriedade Type da tag UL com os seguintes valores

  • Disc: Padrão. Uma lista bola preta totalmente pintada
  • Circle: Uma bola com uma borda preta e sem preenchimento
  • Square: Um pequeno quadrado preto totalmente pintada
